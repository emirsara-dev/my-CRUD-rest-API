# my-CRUD-rest-API

A small CRUD REST API built with Node.js and Express.  
This repository demonstrates basic routing, simple data models, and typical CRUD endpoints for a resource.


## Quick start

1. Clone the repo
```bash
git clone https://github.com/emirsara-dev/my-CRUD-rest-API.git
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm run dev
```

4. Open http://localhost:5000, or the port set by PORT env var.
